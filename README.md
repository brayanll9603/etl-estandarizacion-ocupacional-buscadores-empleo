# etl-estandarizacion-ocupacional-buscadores-empleo
pipeline ETL que permita estandarizar las variables críticas de la base de personas orientadas (ocupación, municipio, nivel educativo, entre otras), alineándolas con catálogos oficiales y reglas de negocio institucionales, con el fin de garantizar calidad, consistencia y estructura analítica para su explotación posterior.
